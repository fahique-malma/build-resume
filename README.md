# build-resume

A Resume Builder web application that enables users to create, edit, preview, and download professional resumes in real-time. The application provides a clean user interface, instant resume preview, local data persistence, and one-click PDF export functionality.

---

## ✨ Features

### 👤 Personal Information
- Add name, email, phone number, location
- Add portfolio website and LinkedIn profile
- Real-time updates in resume preview

### 📝 Professional Summary
- Create a compelling career summary
- Live preview while typing

### 💼 Experience Management
- Add multiple work experiences
- Edit and remove entries dynamically
- Display job title, company, duration, and responsibilities

### 🎓 Education Section
- Add multiple educational qualifications
- GPA and percentage support
- Dynamic updates

### 🚀 Projects Section
- Showcase personal and professional projects
- Add technologies used
- Include GitHub or live project links

### 🏆 Certifications
- Add certifications and completed courses
- Include issuing organization and credential links

### 🛠 Skills Management
- Categorized skills system
- Interactive skill chips
- Add skills using Enter key or comma-separated values

### 📄 PDF Export
- One-click resume download
- Professional A4 PDF format
- High-quality PDF generation using html2pdf.js

### 💾 Local Storage Support
- Automatically saves user data
- Resume remains available after page refresh

## ⚙️ How It Works

### Step 1
Enter your personal information.

### Step 2
Add experience, education, projects, skills and certifications.

### Step 3
View changes instantly in the live preview panel.

### Step 4
Click **Download PDF** to generate a professional resume.

---

## 🎯 Key Functionalities

### Real-Time Rendering
The application updates the resume preview immediately whenever a user modifies data.

### Dynamic Section Management
Users can:
- Add unlimited entries
- Edit existing entries
- Delete unwanted entries

### Automatic Data Persistence
All information is saved using browser Local Storage.

### Professional PDF Export
The resume can be downloaded as a high-quality PDF document.
